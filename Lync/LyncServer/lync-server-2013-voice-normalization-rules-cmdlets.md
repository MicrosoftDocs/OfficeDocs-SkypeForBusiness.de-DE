# <a name="voice-normalization-rules-cmdlets-in-lync-server-2013"></a>Cmdlets für VoIP-Normalisierungsregeln in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-06-20_

VoIP-Normalisierungsregeln werden verwendet, um eine Telefon Wähl Anforderung (beispielsweise wählen von 9 für den Zugriff auf eine Amtsleitung) in das von Microsoft lync Server 2013 verwendete E. 164-Telefonnummernformat zu konvertieren.

<div>

## <a name="voice-normalization-rules-cmdlets"></a>Cmdlets für VoIP-Normalisierungsregeln

Die folgenden Cmdlets können zum Verwalten von VoIP-Normalisierungsregeln eingesetzt werden.

**VoIP-Normalisierungsregeln**

  - <span></span>  
    [Get-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398393(v=OCS.15))

  - <span></span>  
    [New-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398240(v=OCS.15))

  - <span></span>  
    [Remove-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398501(v=OCS.15))

  - <span></span>  
    [Gruppe-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398491(v=OCS.15))

  - <span></span>  
    [Test-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg399003(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [New-CsVoiceRegex](https://technet.microsoft.com/library/Gg412751(v=OCS.15))

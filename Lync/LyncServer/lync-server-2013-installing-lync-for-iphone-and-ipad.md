# <a name="installing-lync-for-iphone-and-ipad-in-lync-server-2013"></a>Installieren von lync für iPhone und iPad in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2014-03-10_

Lync 2013 für iPhone und lync 2013 für iPad sind Benutzer installierbare Anwendungen, die im Apple App Store zur Verfügung stehen.

<div>

## <a name="installing-lync-for-iphone-and-lync-for-ipad"></a>Installieren von lync für iPhone und lync für iPad

Sie können Ihre Benutzer anweisen, lync 2013 für iPhone und lync 2013 für iPad zu installieren, indem Sie Sie von ihren Geräten aus an den App Store weiterleiten. Der App Store für jedes Gerät steht ebenfalls online zur Verfügung.

  - Lync für iPhone steht im App Store unter \< h<span></span>ttp://www.Apple.com/iPhone/from-the-App-Store/zur Verfügung #a0

  - Lync für iPad ist im App Store unter \< HT<span></span>TP://www.Apple.com/iPad/from-the-App-Store/verfügbar #a0

<div>


> [!IMPORTANT]  
> iPhone-Benutzer, die die lync 2013-APP nicht installiert haben und die versuchen, aus einer Besprechungseinladung an einer lync-Besprechung teilzunehmen, werden auf eine Startseite für das Teilnehmer-Startfeld umgeleitet. Diese Seite enthält einen Link zum Installieren der lync 2013-app. Anstatt den Benutzer jedoch an den App Store zu leiten, wird mit diesem Link eine leere Safari-Browser Seite geöffnet. Der Benutzer kann eine von zwei Aktionen ausführen, um dieses Problem zu umgehen: 
> <UL>
> <LI>
> <P>Verwenden Sie die Schaltfläche <STRONG>Start</STRONG> , um die Safari-Seite an den Hintergrund zu senden, und öffnen Sie dann Safari erneut. Wenn die Benachrichtigung "Diese Seite im App Store öffnen" angezeigt wird, tippen Sie auf <STRONG>Öffnen</STRONG> , um im App Store an den lync 2013-Download weitergeleitet zu werden.</P>
> <LI>
> <P>Öffnen Sie den App Store manuell, suchen Sie nach "lync 2013", und laden Sie die APP herunter.</P></LI></UL>

## <a name="verifying-mobile-client-installation"></a>Überprüfen der Installation des mobilen Clients

Nachdem Sie den Client konfiguriert und sich erfolgreich angemeldet haben, verwenden Sie die folgenden Tests, um zu überprüfen, ob Ihre lync-Installation auf Ihrem mobilen Gerät ordnungsgemäß funktioniert.

**Suchen nach einem Kontakt im Unternehmensverzeichnis**

1.  Tippen Sie oben in der Kontaktliste in die Suchleiste, und beginnen Sie mit der Eingabe des Namens eines Kontakts, der nur in der globalen Adressliste (GAL) vorhanden ist.

2.  Stellen Sie sicher, dass der Kontaktname in den Suchergebnissen angezeigt wird.

**Testen der Chat- und Anwesenheitsfunktionen**

1.  Tippen Sie in der Kontaktliste auf einen Kontakt.

2.  Tippen Sie auf der Visitenkarte auf das Symbol **Chat** .

3.  Stellen Sie sicher, dass ein Chatfenster zum Eingeben und Senden einer Sofortnachricht angezeigt wird.

**Testen der Auswahlkonferenz**

1.  Planen Sie in Outlook eine lync-Besprechung.

2.  Öffnen Sie auf dem mobilen Gerät die Besprechungseinladung.

3.  Klicken Sie in der Besprechung auf den Link, um an der Besprechung teilzunehmen.

4.  Nehmen Sie den Anruf vom Konferenzdienst entgegen, und stellen Sie sicher, dass Sie mit dem Besprechungsaudio verbunden sind.

**Testen von Pushbenachrichtigungen**

1.  Registrieren Sie sich auf dem mobilen Gerät von Benutzer a bei lync mit dem Konto von Benutzer a.

2.  Öffnen Sie eine andere Anwendung auf dem mobilen Gerät.

3.  Bei einem anderen Client können Sie sich mit dem Konto von Benutzer B bei lync anmelden.

4.  Senden Sie eine Sofortnachricht von Benutzer B an Benutzer A.

5.  Stellen Sie sicher, dass die Sofortnachricht auf dem mobilen Gerät von Benutzer A erscheint.

# <a name="customizing-the-online-meeting-add-in-in-lync-server-2013"></a>Anpassen des Online Besprechungs-Add-Ins in lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Letztes Änderungsstand des Themas:** 2012-06-28_

Das Online-Besprechungs-Add-in für lync 2013 unterstützt die Besprechungsverwaltung im Outlook-Client für Messaging und Zusammenarbeit. Es gibt mehrere Möglichkeiten, wie Sie das Verhalten des Add-Ins anpassen können. Zu diesen gehören neue Optionen für das Hinzufügen von Logos und Text zum Body of Online-Besprechungseinladungen.

# <a name="planning-for-monitoring-in-lync-server-2013"></a>Planen der Überwachung in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-05_

Der Überwachungsdienst in Microsoft lync Server 2013 bietet Administratoren die Möglichkeit, Nutzungs-, Trend-und Dienst Qualitätsdaten für die Kommunikationssitzungen in Ihrer Organisation zu erfassen. Für die Überwachung in lync Server 2013 ist keine separate Server Rolle mehr erforderlich; Stattdessen ist der Überwachungsdienst in jedem Front-End-Server integriert. Standardmäßig ist die Überwachung in lync Server 2013 jedoch nicht aktiviert. Dieses Dokument hilft Ihnen zu bestimmen, ob die Überwachung in Ihrer Organisation aktiviert werden soll.

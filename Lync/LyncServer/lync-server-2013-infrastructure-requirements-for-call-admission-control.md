<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-08-21_

Zusätzliche Infrastrukturanforderungen wie spezielle Netzwerkrouter müssen nicht erfüllt werden. Durch die Bereitstellung von lync Server 2013 wird automatisch der bandbreitenrichtliniendienst installiert, der zum Erzwingen der Richtlinien für die Anrufsteuerung verwendet wird. Die Anrufsteuerung funktioniert nur mit Clients, auf denen lync verwendet wird.

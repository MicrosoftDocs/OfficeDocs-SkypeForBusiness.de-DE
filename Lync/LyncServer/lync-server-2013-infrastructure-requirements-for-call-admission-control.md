# <a name="infrastructure-requirements-for-call-admission-control-in-lync-server-2013"></a>Infrastrukturanforderungen für die Anrufsteuerung in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-08-21_

Es sind keine zusätzlichen Infrastrukturanforderungen wie spezielle Netzwerkrouter erforderlich. Durch die Bereitstellung von lync Server 2013 wird automatisch der bandbreitenrichtliniendienst installiert, der zum Erzwingen der Richtlinien für die Anrufannahme Steuerung verwendet wird. Die Anrufsteuerung funktioniert nur mit Clients, die lync ausführen.

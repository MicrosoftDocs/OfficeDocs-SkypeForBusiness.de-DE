# <a name="install-prerequisite-software-on-edge-servers-for-lync-server-2013"></a>Installieren der erforderlichen Software auf Edge-Servern für lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-08_

Sie müssen die erforderliche Software auf jeder Edgeserver bereitstellen, die Sie vor der Installation von lync Server 2013 bereitstellen. Dies umfasst die Installation des Betriebssystems auf einem Server, der die Systemanforderungen erfüllt. Ausführliche Informationen zu den Systemanforderungen, einschließlich der unterstützten Betriebssysteme, finden Sie unter [Systemanforderungen für externe Benutzerzugriffs Komponenten für lync Server 2013](lync-server-2013-system-requirements-for-external-user-access-components.md).

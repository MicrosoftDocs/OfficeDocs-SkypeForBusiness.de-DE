# <a name="configure-voice-quality-of-service-for-lync-phone-edition-in-lync-server-2013"></a>Konfigurieren der sprach Quality of Service für lync Phone Edition in lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsdatum des Themas:** 2012-09-29_

Sie können die QoS-Anforderungen (Voice Quality of Service) für lync Phone Edition-Geräte in einem Pool konfigurieren, indem Sie die QoS-Stufe für IP-Telefone festlegen, die eine Verbindung mit lync Server 2013 herstellen.

<div>

## <a name="to-configure-voice-quality-of-service-for-lync-phone-edition"></a>So konfigurieren Sie die Sprachqualität von Service für lync Phone Edition

1.  Öffnen Sie ein Browserfenster, und geben Sie dann die Administrator-URL ein, um die lync Server-Systemsteuerung zu öffnen. Details zu den verschiedenen Methoden, die Sie zum Starten der lync Server-Systemsteuerung verwenden können, finden Sie unter [Öffnen von lync Server 2013-Verwaltungstools](lync-server-2013-open-lync-server-administrative-tools.md).

2.  Klicken Sie in der linken Navigationsleiste auf **Clients**, und klicken Sie dann auf **Gerätekonfiguration**.

3.  Doppelklicken Sie auf der Seite **Device Configuration** in der Liste der Gerätekonfigurationen auf die Konfiguration, deren QoS-Einstellungen Sie ändern möchten.

4.  Geben Sie in **Gerätekonfiguration bearbeiten**unter **Sprachdienst Qualität**die QoS-Ebene an. Die Standardstufe ist **40**.

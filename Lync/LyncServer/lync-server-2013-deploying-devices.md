_**Letztes Änderungsstand des Themas:** 2013-09-10_

Lync Server 2013 umfasst lync-Phone Edition, Software, die auf qualifizierten Geräten ausgeführt wird und traditionelle und erweiterte Telefoniefunktionen, integrierte Sicherheit, Verwaltbarkeit und vieles mehr bietet. Lync Phone Edition funktioniert auf die gleiche Weise wie bei lync Server 2013 mit lync Server 2010. Ausführliche Informationen zum Bereitstellen von Geräten finden Sie unter [Deploying lync Phone Edition](https://go.microsoft.com/fwlink/p/?linkid=285880) in der lync Server 2010 TechNet Library.

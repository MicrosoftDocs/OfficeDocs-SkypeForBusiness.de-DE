# <a name="browser-support-for-lync-server-2013-control-panel"></a>Browser Unterstützung für lync Server 2013-Systemsteuerung

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Letztes Änderungsstand des Themas:** 2012-09-28_

Lync Server 2013 unterstützt die Verwendung des Topologie-Generators und lync Server 2013 Systemsteuerung wird unterstützt, wenn die folgenden Browser verwendet werden:

  - Internet Explorer 10, 32-Bit-Version

  - Internet Explorer 9, 32-Bit-Version

  - Internet Explorer 8, 32-Bit-Version

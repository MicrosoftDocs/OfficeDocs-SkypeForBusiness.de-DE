_**Letztes Änderungsstand des Themas:** 2016-05-25_

Bei lync-hybridbereitstellungen können einige ihrer lync-Benutzer lokal und andere Benutzer in lync Online, beide dieselbe Domäne verwenden. Dies kann die Bereitstellung von lync-Diensten für Benutzer Ihrer Organisation an unterschiedlichen geografischen Standorten oder Benutzern erleichtern, die eine Remoteverbindung herstellen. Sie können auch lync-Hybridkonfigurationen als Migrationspfad zu Microsoft 365 nutzen.

In diesem Abschnitt werden Hybridkonfigurationen für lync Server 2013 beschrieben.

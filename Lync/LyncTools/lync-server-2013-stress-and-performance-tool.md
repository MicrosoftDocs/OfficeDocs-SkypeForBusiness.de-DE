_**Letztes Änderungsstand des Themas:** 2013-01-25_

Das lync Server 2013 Stress and Performance Tool enthält Tools, die die Kapazitätsplanung für lync Server 2013 vereinfachen. Das Tool für lync Server 2013 Stress und Leistung hilft Ihnen dabei:

  - Vereinfachen Sie Ihre Hardwareplanung für lync Server 2013.

  - Verschaffen Sie sich mehr wissen und bewährte Methoden für die Leistungsoptimierung.

  - Messen Sie die Leistung der geplanten lync Server 2013 Bereitstellungen.

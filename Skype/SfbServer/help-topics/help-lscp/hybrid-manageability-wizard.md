# <a name="hybrid-manageability-wizard"></a>Hybridverwaltungs-Assistent

Sie können den Assistenten für die Hybrid Verwaltbarkeit verwenden, um einige der Konfigurationsaufgaben auszuführen, die zum Einrichten einer hybridbereitstellung mit Skype for Business Online erforderlich sind. Weitere Informationen dazu, wie Sie Ihre Bereitstellung für Hybrid konfigurieren, finden Sie unter [Konfigurieren Ihrer Umgebung mit dem Assistenten für die Hybrid Verwaltbarkeit](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).

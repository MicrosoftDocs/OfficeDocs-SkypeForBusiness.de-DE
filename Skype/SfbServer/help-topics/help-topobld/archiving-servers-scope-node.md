# <a name="archiving-servers-scope-node"></a>Archivierungsserver – Bereichsknoten
 
Im Topologie-Generator können Sie Aufgaben im Zusammenhang mit einem einzelnen Archivierungsserver ausführen, indem  Sie entweder in der Konsolenstruktur mit der rechten Maustaste auf den Archivierungsserver klicken, **in** der Symbolleiste auf "Aktion" klicken oder im Aktionsbereich auf eine Aufgabe klicken und dann auf die folgende Archivierungsserveroption klicken.
  
- **Neuer Archivierungsserver**: Dient zum Starten des Assistenten **Neuen Archivierungsserver definieren**, in dem Sie die Einstellungen für den neuen Archivierungsserver angeben können, z. B. den vollqualifizierten Domänennamen, die SQL Server-Archivierungsdatenbank, den Dateispeicher für die Archivierung sowie die Poolzuordnung.
    
> [!NOTE]
> Der Knoten "Archivierungsserver" enthält einen Archivierungsserver, wenn Sie der Topologie im Topologie-Generator zuvor einen Archivierungsserver hinzugefügt haben. Chats oder Webkonferenzen (Messaging) können jedoch erst archiviert werden, wenn Sie auch den Archivierungsserver eingerichtet haben, der die Installation von Skype for Business Server auf dem Server umfasst, den Sie als Archivierungsserver verwenden möchten, und die Konfiguration der Unterstützung für die Archivierung. 
  
Zusätzlich zu den zuvor in diesem Thema beschriebenen Archivierungsserveroptionen stehen im Knoten "Archivierungsserver" auch allgemeine Optionen zur Verfügung, auf die Sie auch über andere Knoten der Konsolenstruktur zugreifen können. Dies umfasst Folgendes:
  
- **Topologie**. Verwenden Sie diese Option, um eine neue Topologie zu definieren, eine vorhandene Topologie zu öffnen, die aktuelle Topologie herunterzuladen, eine Kopie der Topologie zu speichern, die Topologie zu veröffentlichen, eine Datenbank zu installieren, Komponenten zusammenführen oder die Bereitstellung zu entfernen.
    
- **Ansicht**: Mithilfe dieser Option können Sie die Ansicht des Topologie-Generators anpassen, indem Sie die Elemente auswählen, die in den Ansichten der Konsolenstruktur ein- bzw. ausgeblendet werden sollen.
    
- **Hilfe**: Über diese Option greifen Sie auf die Hilfe für den aktiven Knoten zu.

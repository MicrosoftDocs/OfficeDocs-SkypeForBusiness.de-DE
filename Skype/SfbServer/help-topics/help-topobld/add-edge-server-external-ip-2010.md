# <a name="add-edge-server-external-ip-2010"></a>Hinzufügen der externen IP-Adresse des Edgeservers – 2010

Sie können die Eigenschaften auf dieser Seite so konfigurieren, dass der Edgeserver auf die externen Speicherorte des Zugriffs-Edgediensts, des Webkonferenz-Edgediensts und des A/V-Edgediensts verweisen soll.

- Geben Sie unter Externe **IPv4-Adressen** die externen IPv4-Adressen des Edgeservers im **Zugriffs-Edgedienst,** **im Webkonferenz-Edgedienst** und im **A/V-Edgedienst ein.**

- Klicken Sie auf **OK**, um die Änderungen des Topologiedokuments zu speichern und zu übernehmen.

- Klicken Sie auf **Abbrechen**, um die Änderungen zu verwerfen und den Assistenten **Server dem Edgepool hinzufügen** zu schließen.

- Klicken Sie auf **Hilfe**, um dieses Hilfethema anzuzeigen.

Weitere Informationen zum Hinzufügen von Edgeservern zu einer vorhandenen Topologie finden Sie unter [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Weitere Informationen zum Migrieren von Edgeservern finden Sie in [Phase 5: Hinzufügen des Lync Server 2013-Vorschau-Edgeservers zum Pilotpool.](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)

# <a name="add-front-end-associations-2010"></a>Hinzufügen von Front-End-Zuordnungen – 2010

Sie können die Unterstützung für bestimmte Features aktivieren, die die Bereitstellung von anderen Servern erfordern, durch die Server-Rollen mit dem Front-End-Pool zu diesem Zeitpunkt zuordnen. Sie können Serverrollen auch zu einem späteren Zeitpunkt dem Front-End-Pool zuordnen. Die folgenden Serverrollen können einem Front-End-Pool zugeordnet werden:

1. Archivierungsserver. Ausführliche Informationen zum Planen und Implementieren von archivierungsunterstützung, [Your Organization definieren Requirements for Archiving](https://technet.microsoft.com/library/ce0fc0f6-7704-4b80-bf19-a1fa9818fc7a.aspx) in der Planungsdokumentation und [definieren und konfigurieren Sie einen Front-End-Pool oder Standard Edition-Server](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) und [finden bereitstellen Archivierung](https://technet.microsoft.com/library/a89edd16-12d5-4602-ad2f-194b47d1188e.aspx) in der Bereitstellungsdokumentation.

2. Monitoring Server. Ausführliche Informationen zur Unterstützung für die Überwachung implementieren finden Sie unter [Planning for Monitoring](https://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx) in der Planungsdokumentation und [definieren und konfigurieren Sie einen Front-End-Pool oder Standard Edition-Server](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) and [Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) in der Bereitstellung Dokumentation.

3. A / V-edgedienst. Ausführliche Informationen zum Implementieren eines A / V-Edgeserver-service, finden Sie unter [Übersicht über die der Zugriff durch externe Benutzer](https://technet.microsoft.com/library/97aded6c-5fa3-4225-95a6-9ad094d61654.aspx).

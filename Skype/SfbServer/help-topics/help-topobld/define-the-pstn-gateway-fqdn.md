# <a name="define-the-pstn-gateway-fqdn"></a>Definieren des FQDN des PSTN-Gateways

Sie geben den vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN) des PSTN-Gateways (Public Switched Telephone Network) an, mit dem Ihre Enterprise-VoIP-Komponenten mit einem PSTN oder einer PBX (Private Branch Exchange) verbunden werden.

Ausführliche Informationen finden Sie unter [Komponenten der PSTN-Konnektivität in Skype for Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)

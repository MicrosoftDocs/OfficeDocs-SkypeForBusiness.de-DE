# <a name="remove-certificate-invoke"></a>Entfernen eines Zertifikats (Aufruf)
 
Auf **der Seite "Befehle ausführen"** finden Sie Details zum Entfernen des Zertifikats. Lesen Sie die Zusammenfassung zum Entfernen des Zertifikats durch. Stellen Sie sicher, dass die Protokolldatei "Zertifikat entfernen" in  der Dropdownliste angezeigt wird, und klicken Sie dann auf "Protokoll anzeigen", um die Details zum Entfernen des Zertifikats zu überprüfen. Wenn Sie die Zusammenfassung und die Protokolldatei überprüft haben, klicken Sie auf **"Fertig stellen".**

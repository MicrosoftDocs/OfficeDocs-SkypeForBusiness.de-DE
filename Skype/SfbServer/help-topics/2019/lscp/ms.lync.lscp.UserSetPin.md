# <a name="set-pin"></a>Festlegen der PIN

Wenn Sie die PIN des Benutzers generieren möchten, wählen Sie **automatisch eine gültige Pin generieren**aus.

Wählen Sie **Bestimmte PIN von Hand eingeben** und geben Sie anschließend eine PIN in das Textfeld ein, die die in Ihren PIN-Richtlinieneinstellungen angegebenen PIN-Anforderungen erfüllt.

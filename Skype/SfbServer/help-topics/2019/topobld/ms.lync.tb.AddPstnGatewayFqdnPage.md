# <a name="define-the-pstn-gateway-fqdn"></a>Definieren des FQDN des PSTN-Gateways

Sie geben den vollqualifizierten Domänennamen (Fully Qualified Domain Name, FQDN) des PSTN-Gateways an, das Ihre Enterprise-VoIP-Komponenten mit einer PSTN- oder Nebenstellenanlage (Private Branch Exchange, PBX) verbindet.

Weitere Informationen finden Sie unter ["PSTN-Verbindungskomponenten" in Skype for Business Server.](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)

# <a name="associate-av-mcu-with-front-end"></a>Zuordnen der A/V MCU zum Front-End
 
Für alle Enterprise Edition-Front-End-Pools mit aktivierter Konferenz muss ein a/v-Konferenzdienst vorhanden sein, oder Sie müssen einen eigenständigen a/v-Konferenz Pool verwenden. Alle vorhandenen Enterprise Edition-Front-End-Pools, die derzeit über einen a/v-Konferenzdienst verfügen, können ausgewählt werden, um stattdessen diesen neuen eigenständigen a/v-Konferenz Pool zu verwenden.

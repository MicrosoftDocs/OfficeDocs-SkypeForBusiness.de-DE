# <a name="publish-topology-reset-pool-page"></a>Veröffentlichen der Topologie – Seite „Pool zurücksetzen“
 
Dies ist vorläufige Dokumentation und kann geändert werden. Leere Themen wurden als Platzhalter hinzugefügt.
  
Um einem vorhandenen Pool neue Computer hinzuzufügen, müssen Sie den Pool zurücksetzen, damit Windows Fabric die Verbindungen zwischen allen Mitgliedern des Pools neu definieren kann. Wählen Sie dazu den Pool aus, dem Sie Server hinzugefügt haben, und klicken Sie dann auf **"Weiter".**

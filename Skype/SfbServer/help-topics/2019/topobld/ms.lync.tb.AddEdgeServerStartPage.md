# <a name="add-edge-server-support"></a>Hinzufügen der Edgeserverunterstützung

Sie können die Unterstützung für den Zugriff externer Benutzer bei der Bereitstellung Ihrer anfänglichen Topologie oder zu einem späteren Zeitpunkt hinzufügen. Ausführliche Informationen zum Hinzufügen von Edge-Servern zu einer vorhandenen Topologie finden Sie unter [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) in der Dokumentation zur Bereitstellung der Edge-Server.

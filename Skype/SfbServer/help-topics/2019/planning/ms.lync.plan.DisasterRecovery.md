# <a name="disaster-recovery-planning-tool"></a>Disaster Recovery (Planning Tool)
 
Skype for Business Server umfasst Disaster Recovery-Maßnahmen, indem es Ihnen ermöglicht, Front-End-Pools in zwei Rechenzentren zu koppeln.
  
Details zum Vorbereiten Ihrer Umgebung für die Disaster Recovery finden Sie unter [Planen der Hochverfügbarkeits-und Disaster Recovery in Skype for Business Server](../../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md).

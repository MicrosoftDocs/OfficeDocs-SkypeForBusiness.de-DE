# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Anrufqualitäts-Dashboard für Skype for Business Server
 
**Zusammenfassung:** Erfahren Sie mehr über das Anrufqualitätsdashboard, bei dem es sich um ein Berichterstellungstool für Skype for Business Server handelt.
 
Das Skype for Business Server Call Quality Dashboard (CQD) ist eine Berichtsebene, die über der Quality of Experience Database im Monitoring Server in Skype for Business Server und Lync Server 2013 steht. CQD verwendet Microsoft SQL Server Analysis Services, um aggregierte Informationen zur Nutzung und Anrufqualität sowie zum Filtern und Pivotieren des Datensets zur Verfügung zu stellen.
  
Die folgenden Themen sind in diesem Abschnitt enthalten und sollten vor der Bereitstellung des Anrufqualitätsdashboards überprüft werden.

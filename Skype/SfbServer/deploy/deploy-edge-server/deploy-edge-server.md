# <a name="deploy-edge-server-in-skype-for-business-server"></a>Bereitstellen des Edgeservers in Skype for Business Server
 
**Zusammenfassung:** Erfahren Sie, wie Sie einen Edgeserver oder Edgepool in Ihrer Skype for Business Server-Umgebung bereitstellen.
  
Gründe für die Bereitstellung eines Edgeservers oder Edgepools in Ihrer Skype for Business Server-Umgebung Dies ist erforderlich, wenn externe Benutzer, die nicht im internen Netzwerk Ihrer Organisation angemeldet sind, mit internen Benutzern interagieren müssen. Diese externen Benutzer können authentifizierte und anonyme Remotebenutzer, Verbundpartner oder andere mobile Clients sein.
  
## <a name="deployment-checklist-for-the-edge-for-skype-for-business-server"></a>Prüfliste für die Bereitstellung für edge, für Skype for Business Server

Wie bereits erwähnt, geht vieles in eine Edgeserverbereitstellung für Skype for Business Server ein. Diese Prüfliste enthält eine Übersicht über die Aufgaben, die Sie ausführen müssen, sowie Links zu ausführlicheren Schritten.
  
Wir hoffen, dass Sie im Abschnitt "Planen von [Edgeserverbereitstellungen in Skype for Business Server" begonnen](../../plan-your-deployment/edge-server-deployments/edge-server-deployments.md) haben. Wenn nicht, sind viele der Dinge, auf die wir uns beziehen, dort ausführlich. Der Bereitstellungsabschnitt enthält nur Verfahren. Wenn Sie also die Gründe für diese Schritte kennen möchten, sollten Sie mit der Planung beginnen.

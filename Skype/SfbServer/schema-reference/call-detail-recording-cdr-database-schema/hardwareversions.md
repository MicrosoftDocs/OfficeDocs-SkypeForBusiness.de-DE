# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>Tabelle "HardwareVersions" in Skype for Business Server 2015
 
Die Tabelle "HardwareVersions" ist eine Tabelle mit Unterstützung. In jedem Datensatz werden Informationen zu einer Gerätehardwareversion gespeichert.
  
|**Spalte**|**Datentyp**|**Schlüssel/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |Primary  <br/> |Eindeutige Zahl, die diese Hardwareversion identifiziert.  <br/> |
|**Version** <br/> |nvarchar(256)  <br/> | <br/> |Hardwareversion.  <br/> |
